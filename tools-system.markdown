## build
`nox system build`
```shell
Usage:
    build

Description:
    用于生成自动补全脚本

Option:
    --help|-h:                                          -- 使用帮助
    --debug|-x:                                         -- 调试模式
    --slient|-s:                                        -- 编译后不打印 Logo
```
---

## create
`nox system create`
```shell
Usage:
    create --dir <dirname>
    create --script <scriptname>

Description:
    用于 nox 脚本开发，创建一个子目录或脚本

Option:
    --help|-h:                                          -- 使用帮助
    --debug|-x:                                         -- 调试模式
    --dir|-d:                                           -- 创建目录
    --script|-s:                                        -- 创建脚本

```
